    二元一次不定方程求解核武器——辗转相除法的C语言实现¶

    定义（摘自百度）¶

    不定方程一般指含有两个或两个以上未知数的方程，一般具有无数个解

    辗转相除¶

    首先，先了解一下辗转相除法的工作原理（以下内容均为原创）

    例题：17x+23y=51

    不难看出，此题通过奇偶性，整除法，末尾数字均不容易找出特解，比较好的解法是欧拉分离法，但这里要讲另一种解法：辗转相除法。

    一、设17m+23n=1，观察可得x=51m;y=51n

    二、取x、y系数中较大数除以较小数，再以除数除以余数，直至余数为一为止，以下是示例：

    23÷17=1……6;

    17÷6=2……5；

    6÷5=1……1；

    三、倒推，写为1=23*某数±17*某数的形式

    1=6-5*1

    1=6-（17-2*6）*1

    1=3*6-17

    1=3*（23-17）-17

    1=3*23-4*17

    ∴m=-4,n=3

    可得一组特解：x=-4*51=-204

    y=3*51=153

    C语言实现（注：以下只适用于有整数解的情况）¶

    思路¶

    通过程序模拟辗转相除的过程，前半段的数据可用while循环来控制，真正的难点在后半段

    后半段我们最终是要拆成x的系数*某数±y的系数*某数的形式，观察可得，比较好求的是前半部分，也就是正的部分。

    一组数显然不够，我们两组对比来看，此次我们只关注正的那半部分。 1=6-5*1||||||1=5-4*1

    1=6-（17-2*6）||||||| 1=5-（9-5）

    1=3*6-……||||||1=5*2…………

    1=3*（23-17）……||||||1=2*（23-…………）

    1=3*23-……||||||1=2*（87-……）

    观察可得，最终的结果我们只需算到第三步即可，同时，观察第三步，可得正半部分的系数为1+最后一次的商*倒数第二次的商。

    有了前半部分，后半部分就可直接代入求得

    观察也可发现当辗转相除的次数为奇数时，求得的为系数较大数的解，次数为偶数时，求得的为系数较小数的解

    代码实现¶

    讲了那么多思路，现在正式上代码。

    这里提供的代码为c++编写

    法一：通过数组存储每次的数据，缺点为空间复杂度较大（这里是提前开了大数组，也可以用指针）

    这是代码及解释

    //这里就不写头文件了

    int ent[10000000];

    //声明数组，存储商，这里统一用英文后三位来命名

    int main(){

    int x,y,m,n,c,ing;

int jm,jn,m1,n1;

int num=0;   //num为计数变量

cin>>x>>y>>c;

if(x<0&&y<0){

    x-x;

    y=-y;

    c=-c;

}

//避免余数为负

if(x>y){

    m1=m=x;

    n1=n=y;

}

else{

    m1=m=y;

    n1=n=x;

}

//为辗转相除准备

while(ing!=1){

    num++;

    ent[num]=m/n;

    ing=m%n;

    m=n;

    n=ing;

}

//辗转相除过程，为方便数组跳过了ent[0]

if(num%2==1){   //奇偶性判断

    jm=ent[num]*ent[num-1]+1;

    jn=(1-jm*m1)/n1;

}

else{

    jn=ent[num]*ent[num-1]+1;

    jm=(1-jn*n1)/m1;

}

//模拟倒推过程

if(x>y){

    cout<<"x="<<jm*c<<endl;

    cout<<"y="<<jn*c<<endl;

}

else{

    cout<<"x="<<jn*c<<endl;

    cout<<"y="<<jm*c<<endl;

}

//输出

return 0;

    }
